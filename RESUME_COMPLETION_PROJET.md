## 🛠️ Outils et Scripts Créés

### Scripts Python
1. **convert_markdown_to_json.py** - Convertit les données Markdown en format JSON
2. **validate_data.py** - Valide les données de dépôts selon les critères
3. **update_stars.py** - Met à jour les métadonnées via l'API GitHub
4. **discover_repositories.py** - Découvre automatiquement de nouveaux dépôts
5. **discover_manual.py** - Découverte manuelle avec requêtes spécifiques
6. **merge_all_discovered.py** - Fusionne les dépôts découverts dans la base principale

### Fichiers de Configuration

- **metadata.json** - Métadonnées et statistiques du projet
